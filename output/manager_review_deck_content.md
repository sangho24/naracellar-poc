## Slide 1 / 3 — As-Is: 나라셀라의 진짜 문제는 매출이 아니라 현금흐름입니다

> **재프레이밍 포인트**: 기존 deck은 "축소 시장 + 재고 + 채널 사일로"를 병렬로 나열했음.
> 매니저 review용은 **"부채 316억 vs 현금 44억"을 톱 메시지**로 올리고, 나머지는 그 원인으로 종속시킴.
> 컨설팅 매니저급은 매출 성장 스토리보다 "어떻게 현금을 지킬 것인가"에 더 반응함.

### 페이지 헤더
- Eyebrow: `As-Is`
- 제목: **현금흐름 위기 — 매출보다 시급한 문제**
- 부제: 부채 316억 vs 현금 44억, 재고에 자본이 묶여 있고 환·대손 리스크가 누적

### 본문 — 3-column 레이아웃

#### 좌측 column: 현금흐름 압박 (메인 메시지, 가장 큰 비중)
- **부채 316억 vs 현금 44억** — 7배 차이
- **이자 부담 연 16억** — 재고 차입 의존 구조
- **재고에 자본 묶임**: 재고보유일수 **348일** (2022년 215일 → +133일)
  - 미착상품 **102억** (전체 재고의 25%)
  - 재고평가충당금 2.86억
- **2025.3Q 영업손실 2.3억**

#### 중앙 column: 누적되고 있는 부수 리스크
- **환리스크**: 외화부채 73억 → 5% 변동 시 3.6억 손익 영향 (수동 헤지)
- **매출채권 연체**: 25.4억 (연체율 18.6%), 대손충당금 매년 +2억
- **얼로케이션 압박**: 한국에서 안 팔리면 와이너리가 물량 회수 → 포트폴리오 약화

#### 우측 column: 이 문제들의 공통 원인 — 데이터 부재
- **채널 사일로**: 영업1(Off) / 영업2(On) / 리테일 → 각자 자기 데이터만 봄
- **계열사 8개 미연결**, 매장 10여개 POS 연동 불명확
- **프로모션 ROI 측정 도구 부재** — "감" 기반 의사결정
- **상위 4 브랜드군 = 매출 93%** | 161 브랜드·1,955 SKU 중 대부분 재고에 잠김

### 하단 강조 박스 (full width)
> 매출은 시장 축소(3년 32%↓)와 함께 줄어들었지만,
> **진짜 위협은 "줄어드는 매출로 늘어나는 재고를 떠받치고 있다"는 점.**
> 데이터·자동화·AI로 셋 다 해결 가능 — 다음 페이지에서 경로 제시.

### Speaker Note (매니저 설명용)
> "1장은 cash flow 한 장입니다. 매니저님, 나라셀라 IR에서 가장 자주 등장하는 단어가 '재무구조 개선'이에요. 시장은 축소되고 있는데 재고는 늘고 있고, 그 재고를 떠받치느라 차입이자가 연 16억 나가는 구조죠. 그런데 그 모든 원인이 결국 '데이터로 못 푸는 채널 사일로'라는 한 가지로 수렴한다는 게 우리 진단입니다."

### 출처/근거
- 부채/현금/이자: 2025년 3분기 보고서 (DART, 잠정치)
- 재고보유일수: 회계 추정 (2025.3Q 기준)
- 환리스크/대손: 1차 deck 기존 수치
- *Open question (매니저 자문): "이 수치들 중에 매니저님 본부에서 더 신뢰성 있게 받을 수 있는 데이터 있을지?"*

## Slide 2 / 3 — Roadmap: 18개월, 3 Phase, 순서가 중요합니다

> **재프레이밍 포인트**: 기존 deck slide 8을 거의 그대로 가져오되,
> 매니저에게 "이 순서가 왜 중요한가"를 강조하는 카피로 다듬음.
> 각 Phase의 cash flow 효과를 보조 라벨로 추가.

### 페이지 헤더
- Eyebrow: `Roadmap`
- 제목: **기초공사 → 골조 → 인테리어**
- 부제: 순서를 지켜야 무너지지 않습니다. 18개월, 3 Phase, 각 Phase에서 cash flow 효과가 누적됩니다.

### 본문 — 3 Phase 카드 (가로 배열)

#### Phase 1 (0~6개월) — 클라우드 + 데이터 기반 구축
**핵심 작업**:
- 클라우드 ERP 전환 (SAP ByDesign / 더존 ClouD) — 본사 + 물류 + 미국법인 + 직영매장
- SKU×채널×거래처 데이터 정제·축적 (체류기간 측정 가능한 형태)
- 거래처 네트워크 맵핑 (도매상-업장-상권 관계)
- 프로모션 ROI 측정 (Google CausalImpact 기반)

**Cash flow 효과 (Phase 1 시점)**:
- 직접 절감은 작음. 핵심은 **"근거 있는 의사결정"의 발판 마련**
- 클라우드 전환으로 IT CapEx → OpEx 전환 시 IT 관리비 ~30%↓ (벤치마크: 애터미)

#### Phase 2 (6~12개월) — 자동화 + 시뮬레이션
**핵심 작업**:
- 수요예측 모델 (SKU × 채널 단위, 시즌성·환율·트렌드 반영)
- 환리스크 자동 추적 (발주 → 결제까지, 임계치 초과 시 헤지 알림)
- 거래처 신용스코링 (주문 패턴·결제 이력·연체 빈도)
- 채널 배분 What-If 시뮬레이션

**Cash flow 효과 (Phase 2 시점)**:
- 재고보유일수 **348일 → 250일** 목표 ⇒ 약 **100억 재고 감소**, 이자 **연 4~5억↓**
- 환리스크: 연 **1~2억** 절감
- 신용스코링: 대손충당금 증가 추세 억제, 연 **2~3억** 절감
- **이 시점에서 영업손실 흑자 전환 가능**

#### Phase 3 (12~18개월) — AI 개인화 추천 Agent
**핵심 작업**:
- 거래처별 맞춤 와인 추천 AI (1,955 SKU 전체 추천 대상화)
- 영업사원 미팅 브리핑 자동 생성
- 거래처 매출 개선 컨설팅 데이터 (포지셔닝: 공급자 → 성장 파트너)
- 신규 브랜드 론칭 최적화

**Cash flow 효과 (Phase 3 시점)**:
- **Long Tail 매출 +5~10%** (와인앱 개인화 장바구니 +41% 벤치마크 대비 보수적 추정)
- 거래처 차별화 → 가격 협상력 유지

### 하단 강조 박스
> 핵심은 **"Phase 1 없이 Phase 2가 불가능하고, Phase 2 없이 Phase 3은 모래성"**이라는 점.
> 데이터 → 자동화 → AI 순서는 협상이 아닌 물리적 종속관계.

### Speaker Note
> "2장은 매니저님이 가장 의견 주실 부분입니다. Phase 1만 6개월인 게 길어 보일 수 있는데, ERP 전환이 핵심이라 줄이기 어려워요. 다만 '얼마부터 effect가 나오나'에 대한 답은 Phase 2 끝 무렵 흑자 전환이라는 게 우리 추정이고, 매니저님이 보시기에 이 timing이 client 입장에서 sellable한지 듣고 싶습니다."

- 벤치마크 (이마트·애터미·와인앱): 1차 deck slide 12 footnote

## Slide 3 / 3 — Expected Impact: 비용절감만으로 흑자 전환, 성장은 보너스

> **재프레이밍 포인트**: 기존 deck slide 12를 cash flow 순서로 재정렬.
> Long Tail 매출 성장은 마지막에 배치(보조 메시지).
> "지금 영업손실 2.3억 → 비용절감만으로 흑자"를 톱 라인으로 강조.

### 페이지 헤더
- Eyebrow: `Expected Impact`
- 제목: **비용절감만으로 흑자 전환, 성장은 그 위에 얹는 것**
- 부제: 2025.3Q 영업손실 2.3억 → Phase 2 종료 시점에 비용절감 합계가 손실을 넘어섭니다.

### 본문 — 메인 테이블 (cash flow 효과 순서)

| 항목 | 현재 | 전환 후 | 연 효과 | 누적 시점 |
|------|------|---------|---------|-----------|
| **이자 부담 절감** (재고 감소) | 재고 348일, 이자 16억 | 250일 | **연 4~5억↓** | Phase 2 후반 |
| **환리스크 절감** | 수동 헤지, 73억 노출 | 실시간 추적 + 임계 알림 | **연 1~2억** | Phase 2 중반 |
| **대손 절감** | 매년 2억↑ 증가 추세 | 거래처 신용스코링 + 자동 플래그 | **연 2~3억** | Phase 2 후반 |
| **IT 관리비 절감** | 온프레미스 사일로 | 클라우드 통합 | **~30%↓** | Phase 1 종료 |
| **Long Tail 매출** *(보너스)* | 상위 4 브랜드 = 93% | AI 매칭으로 미들/롱테일 활성화 | 전체 매출 **+5~10%** | Phase 3 |

### 하단 — 핵심 메시지 박스 (강조)
- **단기**: 비용절감 합계 **연 7~10억**으로 현재 영업손실 2.3억 충분히 상쇄 → **흑자 전환**
- **중기**: 재고 100억 감소로 자본 회수 → 재무구조 개선 (부채/자본 비율)
- **장기 (진짜 가치)**: Long Tail 활성화 → 매출 2,500억 경로 재활성화, 와이너리 얼로케이션 선순환

### 우측 하단 작은 박스 — 벤치마크 (illustrative)
- 이마트 사이캐스트: 예측오차 **18%↓**
- 유통업 AI 재고비용: **20%↓**
- 와인앱 개인화 장바구니: **+41%**
- 애터미 클라우드: IT비용 **30%↓**

### Speaker Note
> "3장은 우리가 매니저님께 가장 자신 있게 보여드릴 수 있는 장입니다. 단순히 '매출 늘릴게요'가 아니라 '이미 새고 있는 돈을 막기만 해도 흑자 전환됩니다'라는 메시지죠. Long Tail은 보너스로 두고, 매니저님 의견은 '이 절감 추정치들이 client 앞에서 방어 가능한 수준인지'를 듣고 싶습니다. 너무 보수적인지, 적정한지, 더 conservative하게 낮춰야 하는지."
